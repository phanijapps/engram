# @engram/contracts

TypeScript projections of the accepted Engram contract package.

Generated source files under `src/generated/` come from
`contracts/v1/schemas/engram-v1.schema.json`.

```bash
pnpm --filter @engram/contracts generate
pnpm --filter @engram/contracts typecheck
pnpm --filter @engram/contracts test
pnpm --filter @engram/contracts build
```

Do not edit generated files manually.
