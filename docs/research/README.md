# Research Inbox

Use this folder for the research you have gathered.

## Suggested Format

Create one markdown file per source or theme:

```text
docs/research/
  agent-memory-patterns.md
  vector-graph-hybrid-retrieval.md
  privacy-retention-policy.md
  framework-comparison.md
```

For each note, capture:

- Source link or citation.
- Main claims.
- Useful implementation ideas.
- Risks, limitations, or unresolved questions.
- Relevance to this memory layer.

## Synthesis

When enough notes exist, summarize them into an RFC or ADR instead of letting
the research inbox become the design source of truth.
